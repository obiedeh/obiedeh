## Portfolio Thesis

My work is converging around one systems problem:

```text
physical infrastructure
+ edge inference
+ telemetry
+ retrieval-grounded operations
+ human-in-the-loop review
= deployable AI systems for real-world environments
```

The focus is not generic model demos. The focus is building AI systems that can be inspected, benchmarked, explained, and operated.
